Typography Pairings

Primary Serif: Playfair Display / Cormorant Garamond / Libre Baskerville.
Optional Script Accent: Great Vibes / Parisienne / Allura (for headers only).
Supporting Sans: Inter / Lato / Manrope for UI and captions.

Rules: keep body copy in sans/serif for clarity; reserve script for one or two words (e.g., the brand name) to maintain a premium, uncluttered feel.

Usage Notes

  • Minimum logo height for clarity on mobile: 24px (monogram), 32px (wordmark).
  • Clear space: at least the height of the letter D around the logo.
  • On busy photos, place the encircled monogram (M3) or use a cream badge.
  • For TikTok/Shopee avatars, use the round badge (Deep Rose background, cream mark).
  • Never outline the script wordmark; avoid drop shadows and heavy gradients.
